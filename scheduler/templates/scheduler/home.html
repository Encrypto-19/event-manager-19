Steps to use the website

Step 1

If you don't have an account make one

After creating an account login.

If you are logged in, your username should be displayed in the navbar.

User can log out using logout button.

Step 2

The Calendar logo can be used to navigate to the homepage anytime.

A board is a collection of similar tasks. A link to your boards is in the navbar.

Since you don't have any boads right now create a new one used the add button at bottom right.

After creating a new board user is redirected to My boards page.

Step 3

After creating a new board user can open it using the open button.

On the Board page user can delete or update the board and can create a task for the board.

To create a new task use the add button at bottom right.

After creating a task user is redirected to My boards page.

Step 4

On opening the required board user can see all the tasks created in that board.

Clicking on the task name takes user to the task selected which can be deleted or updated from there.
